# 데이터 분석 방법론

## *주제*

데이터 분석 방법론에 대해서 정리한다.

## *나의 생각*

_배경_
기술의 발달로 인해 , 엄청난 양의 데이터가 기술에 의해 축적이 가능해진 이 시대에 축적한 데이터를 가지고 성찰을 얻으려는 여러 시도가 이루어졌다.

여기에서 성찰을 얻는 과정을 체계적으로 정리를 함으로서 , 효율을 높이려고 하여 데이터 분석 방법론이 중요해졌다.

데이터 분석 방법론은 크게 3가지로 나누어서 볼 수 있다.
- 1. 통계적 방법론
- 2. 데이터 마이닝 방법론
- 3. 빅데이터 방법론
