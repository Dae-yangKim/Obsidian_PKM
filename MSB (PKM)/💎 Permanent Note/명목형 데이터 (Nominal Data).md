# 명목형 데이터 (Nominal Data)

## *주제*

범주형 데이터 중 하나인 명목형 데이터에 대해서 정리

## *나의 생각*

범주형 데이터 중 하나인 명목형 데이터에 대해서 설명하자면 ,

해당 데이터에서 가지는 범주 사이에 , <mark>순서적인 관계를 가지지 않는 경우 해당 데이터를 명목형 데이터라고 한다.</mark>

_Ex_
- 종교
- 성별
